VUE JS

Vue js est un framework javascript.

Rendu déclaratif : Vue s'appuie sur le standard HTML avec une syntaxe de type template qui permet de décrire de manière déclarative la structure HTML tout en étant basée sur un état JavaScript. Réactivité : Vue traque automatiquement tout changement d'état JavaScript et met à jour efficacement le DOM en cas de changement.

Un framework est un ensemble cohérent de concept de bibliothèque et d'outils qui fournissent une structure et des fonctionnalités pour faciliter le développement de logiciels.

Il s'agit aussi d'un cadre de travail qui aide les codeurs à construire des applications plus facilement rapidement et efficacement en fournissant des fonctionnalités de bases, des abstractions de haut niveau et des outils de développement

Ici nous venons de rencontrer quelque chose de nouveau. L’attribut v-bind que vous voyez est appelé une directive. Les directives sont préfixées par v- pour indiquer que ce sont des attributs spéciaux fournis par Vue, et comme vous l’avez peut-être deviné, elles appliquent un comportement réactif spécifique au DOM après rendu. Ici cela veut tout simplement dire : « garde l’attribut title de cet élément à jour avec la propriété message de l’instance de Vue ». Si vous ouvrez votre console JavaScript une nouvelle fois et entrez app2.message = 'un nouveau message', vous verrez l’élément du DOM lié — dans notre cas l’attribut title — se mettre à jour.
